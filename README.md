# SonyDev-Bypass

SonyDev-Bypass is a Windows desktop launcher for syncing the SonyDev catalog and downloading supported bypass packages through a clean, installer-based workflow.

![SonyDev-Bypass Preview](1.png)

## Overview

- Built for Windows 10 and Windows 11
- Modern desktop UI with a lightweight launcher flow
- Live catalog with 100+ supported titles
- Automatic in-app update prompts
- Installer-based setup with Repair support
- Multi-language interface for international users

## Features

- Live catalog sync with a launcher-style workflow
- Clean single-window interface for browsing and starting downloads
- Automatic update prompts on launch, so manual file replacement is no longer required
- Installer and Repair flow for fast recovery if the app stops launching
- Multi-language support in the client UI
- Beta delivery flow for quickly shipping new builds and catalog updates

## Install

1. Download the latest installer from the official release channel.
2. Run the setup and complete the installation.
3. Launch SonyDev-Bypass, choose your destination folder, sync the catalog, and start the package you need.

## Updates

SonyDev-Bypass now handles updates through the app itself. If a new build is available, the launcher will offer the update on startup and the process is completed in a couple of clicks.

## Repair

If the app becomes corrupted or stops launching, run the same Setup file again. The installer will offer a `Repair` option so you can restore the installation without doing a full manual reinstall.
