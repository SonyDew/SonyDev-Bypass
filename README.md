## Overview

SonyDev-Bypass is a lightweight Windows client built around a live remote catalog. Instead of relying on a stale bundled list, the app syncs the current SonyDev index and provides a single desktop flow for browsing available entries and starting downloads.

## Highlights

- Windows 10 and Windows 11 support
- Live catalog sync instead of a static hardcoded list
- Multi-language user interface
- Clean launcher-style workflow
- Straightforward update process with settings preserved between builds

## Getting Started

1. Visit the [official SonyDev page](https://sonydev.de/sdbp).
2. Sign in or create an account if the download section requires it.
3. Download the latest available build from the website or the [GitHub Releases page](https://github.com/SonyDew/SonyDev-Bypass/releases).
4. Launch the application, choose your working folder, sync the catalog, and start the package you need.

## Updating

When a newer build is published, download the latest release and replace the previous files with the new ones. Your existing settings should remain intact.

## Notes

- Use official SonyDev sources only.
- Remote catalog entries may change independently from this repository.
- Some security tools may flag launchers or package managers as suspicious. Verify the file source before running anything locally.
