# Trademarks and Branding

The source code in this repository is licensed under GPL-3.0-or-later, but the project's brand identity is not automatically granted under that license.

The following are reserved branding elements of the official project:

- `SonyDev`
- `SonyDev Bypass`
- official logos
- the application icon in [`SonyDevBypass.App/Assets`](SonyDevBypass.App/Assets)
- official website and release branding

## Allowed

- Truthfully saying that your project is based on or forked from SonyDev Bypass
- Keeping copyright notices required by the GPL
- Linking back to the original repository

## Not Allowed Without Permission

- Shipping a modified build while presenting it as the official SonyDev release
- Reusing the official name, icon, or logo in a way that suggests endorsement
- Publishing altered installers, update channels, or binaries as if they were first-party releases
- Using official domains, support channels, or release branding for an unofficial fork

## Requirement for Forks

If you redistribute a modified version, rename the app, installer, and update channel unless you have written permission to keep the original branding.

This file is a project policy note, not legal advice.
